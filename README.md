# Análise de Sentimentos - Movie Reviews (projeto fictício)

## Descrição do Projeto
Este projeto foi desenvolvido para a Film Junky Union, uma comunidade emergente dedicada a entusiastas de filmes clássicos. O objetivo principal é criar um sistema capaz de filtrar e categorizar automaticamente resenhas de filmes, identificando especificamente avaliações negativas e alcançando uma métrica F1 de no mínimo 0,85 no conjunto de teste.

## Metodologia
O projeto segue uma abordagem estruturada de ciência de dados:

1) Carregamento e Exploração dos Dados

- Importação do dataset imdb_reviews.tsv
- Análise exploratória de dados (EDA)
- Investigação de desequilíbrio de classes

2) Pré-processamento

- Limpeza e normalização de texto
- Tratamento de dados ausentes
- Preparação para modelagem (tokenização, vetorização, etc.)

3) Modelagem

- Treinamento de pelo menos 3 modelos diferentes
- Comparação de performance entre modelos
- Otimização de hiperparâmetros

4) Avaliação e Validação

- Teste com conjunto de dados reservado
- Validação com resenhas customizadas
- Análise comparativa dos resultados


## Dados
Fonte: Dataset IMDB fornecido por Andrew L. Maas et al. (2011) - "Learning Word Vectors for Sentiment Analysis"
